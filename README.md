# Electricity Demand Forecasting ‚Äî Argentine SADI Grid with PatchTST

30-day electricity demand forecast for Argentina's national grid (SADI) using PatchTST, a Transformer-based model for time series, trained on real CAMMESA data (2017‚Äì2026).

## Why PatchTST?

SADI demand has strong dual seasonality (summer and winter peaks) and regular weekly cycles. PatchTST is well-suited because:

- Groups consecutive days into *patches* that capture local patterns (e.g. the working week)
- The attention mechanism connects distant patches (e.g. same month in prior years)
- Computationally efficient for long series like this one (3,407 days)

An alternative like iTransformer would add value if explicit covariates (temperature, humidity) were included as model inputs. With demand-only history, PatchTST is the better choice.

## Limitations

- The model does not anticipate discrete events (blackouts, extreme heat waves, surprise holidays)
- The 2020 pandemic period introduces systematic error if it falls in the test window
- Adding temperature and day-of-week as explicit covariates would likely reduce MAPE by ~2‚Äì3 points
